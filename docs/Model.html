Class: Model

Model

new Model(store, templ)

A class that you can use to create, save, and retrieve Instances. Do not create instances of type Model directly; instead use Store#createClass
Parameters:
Name Type Description
store Store
templ Updraft.ClassTemplate
Properties:
Name Type Description
tableName string the name of the table in which entities of this type will be stored. Cannot begin with underscore.
key string the name of the primary key for this table
keyType Updraft.columnType the type of the primary key for this table
columns Object an object describing the fields of objects of this type
Properties
Name Type Description
key string name of the field
value object
Properties
Name Type Argument Default Description
type Updraft.columnType 'int', 'bool', etc.
key bool <optional>
 false set to true on the field that should be the primary key. Only set one.
index bool <optional>
 false create an index on this field
all Query use to construct a query
See:
  • ExampleModel
Example
var ExampleModel = store.createClass({
  tableName: 'exampleModel',
  columns: {
    keyField: { type: 'int', key: true },
    stringField: { type: 'text' },
    jsonField: { type: 'json' },
    indexedField: { type: 'int', index: true },
    childField: { type: 'ptr', ref: ChildClass }
  }
});

var x = new ExampleModel({keyField: 123});
