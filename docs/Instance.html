Class: Instance

Instance

new Instance()

Instances of this type will have properties for all the columns defined in its Model. You can add additional properties, but they will not be saved. Object will track which properties have changed since the last save. Fields of type 'ptr' will return an object { [table's key]:[value], get: [function returning promise evaluating to object] } Do not create objects of type Instance directly; instead create instances of a Model created by Store#createClass
Source:
  • model.js
See:
  • Model
Example
var store = new Updraft.Store();
var Model = store.createClass(...);
var i = new Model(); // i is an Instance
