ClassFactory

new ClassFactory()

Instances of this type will have properties for all the columns defined in Store#createClass. You can add additional properties, but they will not be saved. Object will track which properties have changed since the last save. Fields of type 'ptr' will return an object { [table's key]:[value], get: [function returning promise evaluating to object] }
Properties:
Name Type Argument Default Description
tableName string the name of the table in which entities of this type will be stored. Cannot begin with underscore.
recreate bool <optional>
 false whether the table should be recreated (dropped) if it exists. All data will be lost.
temp bool <optional>
 false whether the table should be created as temporary (will not persist)
