<readonly> columnType :string

Column types. Ignore the values; only use the keys. Note that these are just column affinities, and technically any value type can be stored in any column type. see https://www.sqlite.org/datatype3.html

   }

Schema

Database schema. The outer keys will be the tables in the database. The values will consist of an object whose keys will be the table's rows and values will be the row's type. It will also have an '_indices' object with all the indices found. Note: tables or indices beginning with underscore or 'sqlite' will be ignored
